# Roadmap as of 22/2/2026

This is a living document. Dates are targets, not promises.

---

## Phase 0 — Foundation (Month 1) 22/2/26 - 22/3/26

**Goal:** Establish the project, flash the device, understand the landscape.

- [X] Acquire OnePlus 6T
- [ ] Flash postmarketOS with Phosh
- [ ] Daily drive as secondary phone for 2+ weeks
- [ ] Document every friction point in the issue tracker
- [ ] Set up public repo with build scripts
- [ ] Publish README, GOOD_ENOUGH, ROADMAP, CONTRIBUTING
- [ ] Set up project blog / dev log
- [ ] Join postmarketOS Matrix channel, begin engaging with community

**Exit criteria:** Device is running, friction points are catalogued, project is public.

---

## Phase 1 — Core Reliability (Months 2–3) 22/3/26 - 22/5/2026

**Goal:** Make the basics rock solid. Calls, texts, data, battery.

- [ ] Phone calls reliable on 4G — no drops, clear audio
- [ ] SMS/MMS send and receive including group messages
- [ ] Mobile data stable with manual 4G/5G toggle
- [ ] Wi-Fi auto-reconnect working
- [ ] Battery life reaching 10+ hours (target 12)
- [ ] Waydroid installed and running
- [ ] WhatsApp functional in Waydroid with notification bridging
- [ ] Basic power management tuning
- [ ] First reproducible build script for the custom image

**Exit criteria:** You could use this phone for calls, texts, WhatsApp, and browsing for a full day without charging.

---

## Phase 2 — Daily Driver (Months 4–6) 22/5/2026 - 22/8/26

**Goal:** Something a patient, tech-curious person could use as their main phone.

- [ ] Camera improvements — sub-2-second shutter, usable daylight photos
- [ ] Bluetooth audio reliable with common headphones and speakers
- [ ] Fingerprint reader working consistently
- [ ] GPS locking within 30 seconds, accurate navigation via OsmAnd
- [ ] Notifications arriving promptly for all key apps
- [ ] Messenger (Facebook) working via Waydroid
- [ ] Notes app preinstalled with encryption option
- [ ] Music/podcast playback working (Spotify web + native player)
- [ ] Polished first-boot experience — setup wizard, default wallpaper, sane defaults
- [ ] Thermals acceptable during normal use
- [ ] First public image release
- [ ] 5–10 external testers recruited

**Exit criteria:** Every Must Have in GOOD_ENOUGH.md is marked ✅ Pass. External testers confirm usability.

---

## Phase 3 — Product (Months 7–12) 22/8/26 - 22/2/27

**Goal:** Something you could hand to a non-technical person with confidence.

- [ ] Branding finalised — name, logo, colour palette
- [ ] Project website live with download page
- [ ] One-click image flasher (Raspberry Pi Imager style)
- [ ] Install guide with photos / video
- [ ] OTA update mechanism (even if basic)
- [ ] Waydroid integration polished — Android apps feel less like a container
- [ ] Nice To Have features prioritised based on tester feedback
- [ ] Community growing — Matrix/Discord, 50–100 testers
- [ ] Apply for NLnet / NGI funding
- [ ] First blog posts / videos gaining traction
- [ ] Begin selling preconfigured phones (small scale, 5–10/month)

**Exit criteria:** A non-technical user can buy a phone, flash it (or buy preflashed), and use it daily without help.

---

## Phase 4 — Scale (Year 2)

**Goal:** Grow beyond a single device, build sustainable revenue.

- [ ] Port to 2–3 additional devices (Snapdragon 845 family first)
- [ ] Explore OEM conversations (Fairphone, PINE64)
- [ ] Enterprise pilot — approach privacy-focused organisations
- [ ] Establish CIC or Ltd for commercial operations
- [ ] Curated app list or simple app store
- [ ] Paid support tier launched
- [ ] Attend FOSDEM and other relevant conferences
- [ ] Regular release cadence established

**Exit criteria:** Multiple supported devices, paying customers, sustainable contributor community.

---

## Long-Term Vision (Year 3+)

- OEM partnership for a purpose-built device
- EU government / enterprise contracts
- Full-time team funded by revenue and grants
- A credible third option in mobile — not for everyone, but genuinely good enough for anyone who wants it
